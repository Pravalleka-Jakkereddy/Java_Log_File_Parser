# Java_Log_File_Parser
A simple Java console-based project that simulates a real logging system. It allows users to generate random log entries, view saved logs, filter logs by level (INFO, WARNING, ERROR), search logs by keyword, and view log statistics.
This project is useful for learning Java File Handling, Exception Handling, Date & Time API, Collections, and Menu-Driven Programming.

Features:
  Generate random logs with timestamp,
  View all logs,
  Filter logs by level,
  Search logs by keyword,
  View log statistics,
  Store logs in logs.txt.
